/* Gule Responsive WordPress Theme */
/*-----------------------------------------------------------------------------------*/

Gule is a responsive theme for WordPress. It uses Twitter's Bootstrap version 3.0. It comes with core HTML, CSS, and Javascript for type, form, grids, navigation, buttons, dropdown menus, and more. Visit http://www.oopthemes.com/ to see the theme in action, to download it, or to read the documentation.

/* Resources used in theme */

1 : Twitter Bootstrap :

Bootstrap is Twitter's toolkit for kickstarting CSS for websites, apps, and more. It includes base CSS styles for typography, forms, buttons, tables, grids, navigation, alerts, and more.

To get familiar with the features -- checkout http://twitter.github.com/bootstrap!

2 : Elusive Ico Fonts :

The Elusive Iconfont is an icons webfont, optimized for use with twitter's bootstrap. https://github.com/aristath/elusive-iconfont

Images used in the demo of the theme and in the screenshot are used from the pixabay.com (http://pixabay.com/en/editors_choice/)
